# AirlineDB-SQL-Case-Studies
SQL analysis project using an Airline Database. Solved real-world business problems using joins, aggregations, date functions, and window functions.
## Project Overview

This project analyzes an airline reservation database consisting of multiple related tables including bookings, tickets, flights, airports, boarding passes, and aircraft information. The goal is to solve business-oriented questions using SQL and demonstrate practical data analysis skills. The database includes entities such as Bookings, Tickets, Ticket Flights, Boarding Passes, Flights, Airports, Aircrafts, and Seats.
